# simple-ether-wallet
[Ethereum入門](http://book.ethereum-jp.net/)でのMeteorによるDapp開発チュートリアル用サンプルプログラム。

simple-ether-walletはEthereumの内部通貨であるEtherのシンプルなwallet（ウォレット）です。アカウント毎のEtherの残高のモニタリングや送金処理を行います。
